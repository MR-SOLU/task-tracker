# Task Tracker

## Project Overview
Task Tracker is a simple application for adding and managing daily tasks.  
This project also demonstrates Git version control concepts such as branching, merging, conflicts, and documentation.

## Installation
1. Clone the repository:
   git clone https://github.com/MR-SOLU/task-tracker.git
2. Go into the folder:
   cd task-tracker
3. Run the project:
   python src/main.py

## Features
- Add tasks
- Update tasks
- Delete tasks
- View all tasks
- Demonstrates Git concepts

## Usage
To run:
python src/main.py
